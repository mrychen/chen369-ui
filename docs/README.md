### 快速开始

### 第一步

#### 1.安装组件库

```bash
<!-- npm -->
npm i chen369-ui
<!-- yarn -->
yarn add chen369-ui
```

#### 2.引用组件库

```javascript
//全部引入
import "chen369-ui/dist/css/index.css";
import Chen369Ui from "chen369-ui";
Vue.use(Chen369Ui);

//按需引入
import "chen369-ui/dist/css/ChenButton.css";
import { ChenButton } from "chen369-ui";
Vue.use(ChenButton);
```
