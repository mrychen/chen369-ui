# 键盘组件

# 使用方式

必须要有一个可输入的元素的id设置为：wbkj_input_text

input type="text" id="wbkj_input_text">
<c-keyboard :isShowKeyboard.sync="isShowKeyboard" @getValue="getValue" />

# 参数

参数名称 说明 默认值 是否必传
isShowKeyboard 是否显示虚拟键盘 true 是
@getValue 传入子元素的函数，供子元素函数调用，用于打印输入文本的值 否
